Javascript Basics

20 January 2016

How does Javascript compare to HTML and CSS?

HTML is the content and most often the base structure of your site. It does not style, or change font, it just tells the site what text is being displayed, what files to load and what language it everything is written in.

CSS controls all the styles of your site. From the fonts used, to the color or your headers and side bars, that is CSS working its magic. With CSS you are very simply defining which HTML element to style, then giving it style options.

Javascript, on the other hand, is all about the interaction with your site. Javascript can do many things, from take a key press and turn it into a change in your HTML, to making complicated chat windows or completing maths problems. With Javascript on your site, the possibilities are extremely open on what you can create!

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'

Control flow relates to the order in which tasks or code is run. For example:

Wake up

Tired => Press snooze

Not tired => Get out of bed

Loops on the other hand, iterate over the same peice of code until the condition turns false. For example:

Go to bathroom and brush teeth

Are teeth clean?

No => Action: Up then down

Yes => Stop Brushing

Explain the difference between accessing data from arrays and object literals

Arrays and object literals can contain items. The difference between them is how those items are stored. With arrays they will just be a list of items: 18, Cuba St, Marketing, however, with object literals we can go a step further to give ID's to each of the object items: Age: 18, Area of work: Cuba St, Ocupation: Marketing.

They also differ in how we access those values. With Object literals, you access their values using dot notation (console.log(person.location);), with arrays, you access the values using numerical index (console.log(array[2]);)

Explain what functions are and why they are useful

In Javascript a function is a block of code that is designed to perform a particular task. These are executed when something calls that function.

For Example, you could set up a key press to trigger a function, that function could do anything, from move a player in a game forward, to submit a for or change a website to a mobile view.
