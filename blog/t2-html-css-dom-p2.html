What's this CSS all about?

What is grid based design?

A grid based design is a reliable way for your website to display information. Using grids you can keep everything in your website aligned and tidy. You can also use this to draw the readers eyes in the desired direction point them towards the information you want them to see.

Whats all the hype about responsive webpages?

The hype is well warranted. Having a responsive website means that your website will work on any device and screen size. This is a huge benefit as often 30% of your website traffic is coming through from smaller devices these days. This is often dont through making certain break point which will adjust your site at different screen widths. This means you can make a table work differently from a phone and ensure you are displaying the most important information.

Semantic structure

Semantic structure is how your website is setup using the right semantics in the right place. For example, you wouldn't use a nav tag for an article. This helps search engines understand what your website is and also helps with accessability. This is an example of semantic structure.
