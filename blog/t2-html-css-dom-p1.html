Describing HTML, CSS and DOM

HTML is the content on your page. Any text, images blogs etc will all be in your HTML.

CSS is the styling that is layed out on top of your content. Without CSS a website is just text with no background and look and feel to it.

DOM is where each element of your websites site. You can think of it as a family tree. This helps when controling elements as some inherit traits, just like people.

Boxify

Boxify is when you put a website up into all the boxes that are used to create it. This is helpful when taking an image or another website and redesigning it.
